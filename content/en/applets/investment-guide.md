## Introduction

This guide provides a comprehensive overview of the Investment Applet, a powerful tool for managing investment portfolios within the Unified Financial System (UTMFin Replacement).

## Investment Module

The Investment Applet is organized into several modules, each designed to handle specific aspects of investment management. The table below provides a high-level explanation of each module:

| Module Name | Description |
| :--- | :--- |
| **Portfolio Management** | Allows users to create, monitor, and manage investment portfolios. |
| **Trade Execution** | Facilitates the buying and selling of financial instruments. |
| **Performance Analysis** | Provides tools for analyzing the performance of investments. |
| **Risk Management** | Helps users identify and mitigate investment risks. |
| **Reporting** | Generates a variety of reports for analysis and compliance. |
