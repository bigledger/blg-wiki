# 库存管理模块

## 概述

BigLedger 库存管理模块提供对整个供应链的全面控制，从采购到仓库管理再到订单履行。凭借实时跟踪、智能预测和跨所有渠道的无缝集成，确保最佳库存水平的同时最小化持有成本。

### 库存策略

#### 库存估值方法

| 方法 | 描述 | 最适用 |
|------|------|--------|
| FIFO | 先进先出 | 易腐商品 |
| LIFO | 后进先出 | 不易腐商品 |
| 平均成本 | 加权平均 | 价格稳定 |
| 具体识别 | 实际成本跟踪 | 高价值物品 |

#### 再订货策略

1. **固定订货量**
   - 每次订购相同数量
   - 当库存达到再订货点时

2. **固定订货期**
   - 定期间隔订购
   - 数量可变

3. **最小-最大**
   - 低于最小值时订购
   - 订购至最大值

4. **经济订货量 (EOQ)**
   - 最优订货规模
   - 最小化总成本
