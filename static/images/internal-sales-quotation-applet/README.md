# Internal Sales Quotation Applet - Screenshot Guide

This directory holds screenshots for the Internal Sales Quotation Applet documentation. **Screenshots must be captured manually** from akaun.cloud and saved here with the filenames below.

## How to reach the applet

1. Open: **https://akaun.cloud/#/bigledger/akaun-platform/container/launchpad-panel**
2. Log in if required.
3. Click the **[Internal sales quotation]** tile on the launchpad.
4. Use the list below to capture each screen and save with the exact filename.

---

## Required screenshots (save to this folder)

| # | Filename | What to capture |
|---|----------|-----------------|
| 1 | `main-listing-page.png` | **Main listing/dashboard** – Sales Quotations list with filters, search, status. (Default view after opening the applet.) |
| 2 | `create-quote-form.png` | **Create form** – Click **"+" / Add New**, then capture the new quotation form (customer, reference, dates, currency). |
| 3 | `edit-quote-form.png` | **Edit form** – Open an existing (draft) quotation from the list; capture the same form with data pre-filled. |
| 4 | `add-quote-items.png` | **Add quote items** – On a quote (create or edit), add a line and capture the item selection, quantity, pricing, discount. |
| 5 | `pending-approvals-page.png` | **Pending approvals** – Open "Pending Approvals" and capture the list of quotes awaiting approval. |
| 6 | `convert-to-order.png` | **Convert to order** – Open an accepted quote and capture the "Convert to Order" action / confirmation. |
| 7 | `quotation-settings-page.png` | **Settings** – Go to **Settings > Quotation Settings** and capture the main settings page. |
| 8 | `pricing-schemes-settings.png` | **Settings** – **Settings > Pricing Schemes** (listing or config). |
| 9 | `approval-settings-page.png` | **Settings** – **Settings > Quotation Approval** (approval workflow config). |
| 10 | `team-reports-page.png` | **Reports** – Team reports / dashboard showing team quotations and filters. |

Save all files into: **`static/images/internal-sales-quotation-applet/`**

---

## Screenshot guidelines

- **Format:** PNG.
- **Size:** Full browser window or main content area (e.g. min 1280px width).
- **Names:** Use the exact filenames in the table (lowercase, hyphens).
- **Content:** Frame the relevant UI; avoid unnecessary browser chrome if possible.
- **Privacy:** Blur or remove any real customer/personal data.

---

## After capturing

The documentation in `content/en/applets/internal-sales-quotation-applet.md` already uses figure shortcodes that reference these paths. Once the PNG files are in this folder, they will show up in the built site.
